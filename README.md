# Portfólio - Hélio Levi

Este é o repositório do meu portfólio pessoal. O objetivo deste projeto é apresentar meu trabalho, minhas habilidades como desenvolvedor Front-end e estudante de Engenharia de Software, e facilitar o contato através das minhas redes sociais.

## 🚀 O que foi feito neste projeto?

Este projeto consiste em um site responsivo e moderno com múltiplas páginas para estruturar minha presença digital:

- **Página Inicial (`index.html`)**: Uma página de apresentação ("Home") focada em introduzir quem eu sou, meu objetivo principal de elevar negócios através da tecnologia, e links diretos para minhas redes sociais (Instagram, GitHub, LinkedIn).
- **Página 'Sobre Mim' (`about_me.html`)**: Uma seção dedicada a contar um pouco mais sobre minha trajetória, missão e visão de mercado, enfatizando a importância de um bom posicionamento digital e branding para empresas.
- **Página de Projetos (`projects.html`)**: Uma página estruturada para exibir os trabalhos e projetos que já desenvolvi.
- **Estilização e Layout (`style/portifolio.css`)**: Uso de CSS nativo para criar um design atraente, trabalhando com flexbox/grid, tipografia adequada e posicionamento de elementos para garantir uma boa experiência de usuário (UX).

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica de todas as páginas.
- **CSS3**: Estilização, layout e responsividade.
- **Tecnologias Básicas**: Criado inicialmente para agrupar projetos e incluir uma breve descrição do usuário.

## ⚙️ Como executar

Por ser um projeto puramente construído com HTML e CSS estático, basta baixar ou clonar o repositório e abrir o arquivo `index.html` em qualquer navegador web de sua preferência.
